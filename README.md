Descripción:
Push_swap es un proyecto del currículo de 42 cuyo objetivo es desarrollar un algoritmo de ordenamiento altamente optimizado que ordene una pila de números utilizando un conjunto limitado de instrucciones predefinidas. El reto consiste en implementar una solución que no solo funcione correctamente, sino que también minimice el número de operaciones ejecutadas.

Objetivos del proyecto:

Diseñar e implementar un algoritmo de ordenamiento eficiente (por ejemplo, variantes de quicksort, mergesort o radix sort adaptadas al contexto del proyecto).

Comprender en profundidad la complejidad algorítmica y cómo optimizar el rendimiento.

Manipular estructuras de datos como pilas (stacks) y listas enlazadas.

Cumplir con las restricciones estrictas de operaciones impuestas por el enunciado.

Operaciones permitidas:

sa, sb, ss — intercambiar los dos primeros elementos de una pila.

pa, pb — pasar el primer elemento de una pila a la otra.

ra, rb, rr — rotar los elementos hacia arriba.

rra, rrb, rrr — rotar los elementos hacia abajo.
